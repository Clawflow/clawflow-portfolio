# ClawFlow Portfolio

Modern portfolio-/landningssida för ClawFlow – AI-automation för svenska SMB-företag.

## Stack
- HTML5
- CSS3 (animationer, glassmorphism, responsiv)
- Vanilla JavaScript (scroll-reveal, cursor-glow, smooth scroll)

## Funktioner
- Sticky navbar med blur-effekt
- Hero med floating cards och animationer
- Stats-sektion med gradients
- Tjänster & case studies
- Kontaktsektion
- Fullt responsiv (mobile-first)
